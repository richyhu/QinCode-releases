### 变更内容

#### 1. 修复 proxyChat.ts HTML 错误解析崩溃

**问题**: 聊天时代理服务器返回 HTML 错误页面（Cloudflare 502、Nginx 错误等），代码直接 `JSON.parse(errorText)` 导致抛出 `Unexpected token '<', "<!DOCTYPE "... is not valid JSON`，用户体验极差。

**修复** (`src/api/proxyChat.ts`):
- 在 `JSON.parse` 前先检查内容是否以 `{` 或 `[` 开头（看起来像 JSON）
- 如果是 HTML，直接截取前 200 字符作为错误信息展示，不再尝试 JSON.parse
- 错误信息格式改为：`HTTP {status} - {preview}`，更清晰

**修复前**:
```
Unexpected token '<', "<!DOCTYPE "... is not valid JSON
```

**修复后**:
```
代理请求: HTTP 502 - <!DOCTYPE html><html>...（前200字符）
```
